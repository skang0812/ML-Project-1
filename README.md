# NHANES & LMF Mortality Prediction Project
**UVA BSDS | Group Project 1**

## Project Overview
This project utilizes the **CDC NHANES (1999-2000)** and **Linked Mortality File** to predict mortality outcomes using machine learning. We implement $k$-Nearest Neighbors ($k$-NN) for both classification (predicting vital status) and regression (predicting months of follow-up).

## AI Use Statement
Generative AI was utilized in this project primarily to assist with repository structure, code debugging for $k$-NN optimization loops, and the formatting of technical documentation.
